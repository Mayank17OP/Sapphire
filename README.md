# Sapphire BurnoutGuard

Sapphire is an AI-powered burnout prevention tool that uses computer vision (simulated in this demo) to detect early signs of digital fatigue and suggests smart interventions.

## Features

- **Real-time Fatigue Detection**: Monitors blink rate and posture (Mocked for demo).
- **Focus Energy Graph**: Visualizes your energy levels over time.
- **Sapphire AI Chat**: A supportive companion for mental health check-ins.
- **Smart Interventions**: Breathing exercises, 20-20-20 rule timer, and focus sessions.
- **Cyberpunk-Wellness Aesthetic**: A unique visual style designed to be calming yet futuristic.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS v4, Custom CSS Variables
- **Animations**: Framer Motion, tsParticles
- **Charts**: Recharts
- **State Management**: Zustand (with localStorage persistence)
- **Routing**: react-router-dom

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Mocked Features

This prototype includes simulated features for demonstration purposes:

-   **Webcam Detection**: The app requests camera permissions to show the "live feed" UI, but the fatigue metrics (blink rate, yawn count) are generated by a deterministic simulation algorithm rather than a real ML model.
-   **Download CTA**: The download button simulates a desktop app download but does not actually provide a file.

-   **Typography**: Montserrat (Headers), Poppins (Body).
